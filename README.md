# 🍔 BurgerFlow

Sistema de gerenciamento para hamburguerias desenvolvido utilizando HTML, CSS e JavaScript.

## Funcionalidades

- Dashboard
- Cardápio Digital
- Controle de Mesas
- Cadastro de Clientes
- Financeiro
- Área Administrativa
- Configurações

## Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Objetivo

Projeto desenvolvido para estudos e composição de portfólio.
